2.1 System preparation

Tested on Debian 10 (Buster).

Install these packages to use Texinfo in our example: 

sudo apt-get install git make texinfo info pinfo texlive-binaries \
   texlive-latex-base texlive-plain-generic

Note: The info(1) and pinfo(1) are viewers of Info output formats (hyper-text format used even before World Wide Web. Similarly we use make(1) as build system and git(1) to checkout our sample project.

Note: We must install texlive-latex-base, because Texinfo uses LaTeX fonts.

Now checkout this project from GitHub using: 

mkdir -p ~/projects
cd  ~/projects
git clone https://github.com/hpaluch/html-and-pdf-generators.git
cd html-and-pdf-generators
